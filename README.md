# BMI Conversion

A CLI app that allows you to calculate BMI from weight and height or calculate weight from BMI and height 

# Note

- Only imperial units are supported. Metric units may be supported based on demand.
- This app can only be used for adult BMI

# Get Started

1. Fork the BMI Conversion repo on Github
2. Use VSCode to get the forked repo onto your machine
3. Install Python3
4. Type ```python3 main.py``` to run the app
